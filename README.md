# EPA.gov (epa-gov)
The U.S. Environmental Protection Agency (EPA) provides APIs and data services for accessing environmental data. The EPA's Central Data Exchange (CDX) system provides APIs for reporting and accessing environmental compliance data including emissions, water quality, and hazardous waste information.
